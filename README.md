## Instalation
Clone the repo onto your computer
```sh
git clone https://github.com/Samhuw8a/Steganography
cd Steganography
```

install the Programm using pip
```sh
python -m pip install .
```

Now you can run the Programm by calling
```sh
steganography -h
```
